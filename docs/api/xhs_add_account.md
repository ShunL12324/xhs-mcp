# xhs_add_account

添加新账号或重新登录已有账号。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | string | 是 | 账号名称 |
| `proxy` | string | 否 | 代理服务器地址 |

## 返回值

```json
{
  "success": true,
  "accountId": "uuid",
  "qrCodeUrl": "https://...",
  "message": "请扫描二维码登录"
}
```

## 登录流程

1. 调用此工具，传入账号名称
2. 系统会启动无头浏览器并捕获登录二维码
3. 返回二维码图片的临时 URL
4. 用小红书 App 扫描二维码
5. 登录成功后会话自动保存

## 示例

### 添加新账号

```
xhs_add_account({ name: "我的账号" })
```

### 使用代理

```
xhs_add_account({
  name: "代理账号",
  proxy: "http://127.0.0.1:7890"
})
```

### 重新登录

如果账号已存在，调用此工具会触发重新登录：

```
xhs_add_account({ name: "已有账号" })
```

## 注意事项

- 二维码 URL 有效期有限，请尽快扫描
- 登录成功后会话持久化到数据库
- 会话过期后需要重新登录
